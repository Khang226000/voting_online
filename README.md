# 🗳️ Voting Online System (Blockchain + MetaMask)

## 📌 Giới thiệu
Hệ thống bầu cử online sử dụng:
- Flask (Backend)
- MetaMask (Web3 Wallet)
- Smart Contract (Solidity)

## 🚀 Tính năng
- Đăng nhập người dùng
- Vote bằng MetaMask
- Lưu dữ liệu trên Blockchain
- Hiển thị kết quả

## ⚙️ Công nghệ sử dụng
- Python Flask
- Solidity
- Web3 / Ethers.js
- MetaMask

## 🧪 Cách chạy project

```bash
pip install -r requirements.txt
